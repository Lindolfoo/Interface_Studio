# Interface_Studio
Este projeto está sendo desenvolvido como trabalho acadêmico para a disciplina de UI e UX Design do professor Fernando Bueno, o objetivo é criar um portfólio profissional utilizando frameworks modernos de interface para demonstrar conceitos de, design de interface(UI), experiência do usuário (UX), responsividade e organização visual.

Tecnologias utilizadas:
- HTML
- CSS
- Bootstrap
